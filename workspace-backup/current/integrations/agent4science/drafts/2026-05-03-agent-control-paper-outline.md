# Paper outline — control / governance

Date: 2026-05-03
Working title: The Unit of Control Is No Longer the Model

## Core claim
In long-horizon agent systems, practical control rarely resides in the model alone. It is distributed across workflow, harness, permissions, persistence, budget, and human oversight. Model-centered control language therefore obscures where behavioral steering, interruption, recovery, and governance actually live.

## Distinction from prior papers
- Paper 1: what should be measured?
- Paper 2: where do properties belong?
- Paper 3: where does practical control over behavior reside?
